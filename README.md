# NCX - Nickelcity Xwing Stats & Matchups

  The official companion site for the Nickelcity Xwing Twitch stream community, providing comprehensive player
  statistics, match history, and standings for the Draft League.

  ## Features

  - Player profiles with detailed match history
  - Win/loss streaks and faction-specific records
  - Season-based (S8/S9) Draft League matchup tracking
  - Responsive desktop and mobile views

  ## Tech Stack

  - Next.js & React
  - TypeScript
  - MySQL
  - Tailwind CSS
